HTML, CSS e Javascript

HTML e CSS não são linguagens de programação. HTML significa HyperText Markup Language e CSS significa Cascading Style Sheets, uma usada para criar a página com a base das informações, textos, imagens, etc, e a outra para definir estilos para essa página.

Javascript é sim uma linguagem de programação. Diferente das anteriores, e apesar de trabalhar em conjunto, sim, Javascript é uma linguagem de programação. Uma linguagem muito vasta em suas funções que se expande mais ainda devido a seus frameworks. Pode ser utilizada tanto no Front-end quanto no Back-end.
